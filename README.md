## 💡 주요 기능
<details>
 <summary><h3 style="display:inline; margin-left:4px">Image</h3></summary>
 
- 이미지 URL을 입력하여 이미지 카드 추가
</details>

<details>
 <summary><h3 style="display:inline; margin-left:4px">Video</h3></summary>
 
- YouTube 영상 URL을 입력하여 비디오 카드 추가
</details>

<details>
 <summary><h3 style="display:inline; margin-left:4px">Note</h3></summary>
 
- 제목과 내용을 입력하여 메모 카드 추가
</details>

<details>
 <summary><h3 style="display:inline; margin-left:4px">Todo</h3></summary>
 
- 할 일 목록 카드 추가
</details>

<details>
 <summary><h3 style="display:inline; margin-left:4px">Drag & Drop</h3></summary>
 
- 카드 순서를 드래그 앤 드롭으로 변경 가능
</details>

<details>
 <summary><h3 style="display:inline; margin-left:4px">Delete</h3></summary>
 
- 각 카드의 X 버튼으로 삭제 가능
</details>
